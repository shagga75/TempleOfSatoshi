üíæ HARDWARE WALLETS

DEFINITION: A hardware wallet is a physical device that serves two functions: securely generating and storing private keys, and signing transactions with the private key to send them. They are called cold wallets because they are never connected to the internet.

ADVANTAGE: You have the option to connect them to your own node without sharing your information with external servers or third parties, thus preventing them from spying on your transactions or how many funds you possess.

WALLET NAME DESCRIPTION SECURITY LEVEL
ColdCard Wallet For advanced users with many available options. There's no need to connect it to a computer as it uses microSD for transactions. HIGH
Trezor One of the most popular devices, along with Ledger. It has 3 different models: One, Model T, and Safe 3. *It does not have a secure element (except for the Safe 3 model) and there is a possibility of extracting the seed with physical access to the device. It's mandatory to add a passphrase to the recovery words. We do NOT recommend using the 'ONE' model. MEDIUM
Ledger Closed-source hardware wallet dependent on its native Ledger Live application. We do not recommend its use for storing Bitcoin. Several personal data hacks of buyers have occurred. High risk of attack. LOW
Bitbox02 Open-source hardware wallet from Switzerland. Ideal for novice and intermediate users. HIGH
Specter DIY A project to make your own hardware wallet. For advanced users. HIGH
Seedsigner A project to make your own hardware wallet focused on multisignatures. For advanced users. HIGH
Passport Hardware wallet created by Foundation Devices. It has the same secure element and a similar interface to Coldcard. Easy to use and versatile. It's wireless and supports QR codes and MicroSD. HIGH

WALLET NAME PLATFORM DESCRIPTION
Samourai Wallet ANDROID Non-custodial wallet with multiple privacy tools (Android only). Allows connection to your own node via Dojo. It also has coinjoin and post-spend tools, and features coin control.
FullyNoded iOS macOS FullyNoded 2 is an open-source wallet that connects via the authenticated Tor V3 service to your own full Bitcoin node. It features coin control.
BlueWallet iOS ANDROID A feature-rich and easy-to-use wallet. Allows connection to your own node via an Electrum server. Integration with Coldcard wallet. Coin control enabled.
Green by Blockstream iOS ANDROID Green is a Bitcoin wallet that provides users with a good combination of security and ease of use. A good option to try it via Testnet.
Sentinel ANDROID A 'watch-only' wallet. It allows you to send funds to that wallet but not spend from it. To do this, you need to import the 'XPUB' (extended public key) of your wallet. A good way to improve your security. It can be connected to a node through Dojo from Samourai Wallet.
